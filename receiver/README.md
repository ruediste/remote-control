# NRF24 Pinout:

![NRF 24](doc/nrf24pinout.jpg)

Arduino Pins:

- PA4: CS
- PA5: SCK
- PA6: MISO
- PA7: MOSI
- PB0: CE
- 3.3V: VCC

# LM293D Pinout

![LM293D](doc/lm293d.png)
Arduino Pins

- pinD1A = PB4;
- pinD1B = PB5;
- pinD2A = PB6;
- pinD2B = PB7;
- pinEN1 = PB3;
- pinEN2 = PB9;
